# Introducción a los problemas de Machine Learning

## Tipos de problemas
1. Según el tipo de aprendizaje
    - **Supervisado**: hay un conjunto de datos etiquetados. Se utiliza para los problemas de clasificación.
    - **No supervisado**: no hay un conjunto de datos etiquetados, y se requiere resolver problemas de:
        + agrupamiento / clasificación (clustering)
        + topic modeling
    - **Semi supervisado**: hay un conjunto de datos etiquetados y no etiquetados
    - **Por refuerzo**: hay un agente que aprende a tomar decisiones (reinforcement learning)
2. Según tipo de predicción (algunos)
    - **Regresión**: se predice un número (continuo)
    - **Clasificación**: se predice una etiqueta, categoría, clase
        + Clasificación binaria
        + Clasificación multiclase ( + de 2 clases)
    - **Recomendación**: se predice una lista de elementos recomendados
    - **Anomalías**: se detectan anomalías en los datos
    - **Proyección (forecasting)**: se predice un valor futuro en una serie temporal
    - **Procesamiento de lenguaje natural**: se predice un valor en un texto

## Métricas

Distintos tipos de problemas requieren distintos tipos de métricas para evaluar el desempeño del modelo.
Evaluar el modelo implica verificar si el modelo entrenado es lo suficientemente preciso.

### Problema de clasificación binario
- Posibles resultados: 0 o 1 (verdadero o falso)
- Errores posibles
    + Falso positivo
    + Falso negativo

- Matriz de confusión
    + 
- Posible enfoques
    + Reducir los falsos positivos
    + Reducir los falsos negativos( por ejemplo clasificar si una persona tiene o no una enfermedad). La medida correcta es el recall.
    $$
    \text{Recall} = \frac{\text{Verdaderos positivos}}{\text{Verdaderos positivos} + \text{Falsos negativos}}
    $$
